# Smart Cooler Firmware — Production IoT System

Python firmware for Raspberry Pi Zero 2W controlling smart evaporative cooling hardware.

## Hardware
- Motor speed control (PWM)
- Solenoid valves & Pump
- AHT20 temperature/humidity sensor
- Safety fallbacks and emergency handling

## Protocols
- I2C, GPIO, MQTT

## Deployment
- 24/7 live production on AWS EC2

## Tech Stack
Python | Raspberry Pi | MQTT | I2C | GPIO | PWM
