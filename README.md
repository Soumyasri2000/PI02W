# Smart Cooler Firmware — Production IoT System
Python firmware for Raspberry Pi Zero 2W controlling 

smart evaporative cooling hardware.

Hardware: Motor (PWM), Solenoid valves, Pump, AHT20 sensor

Protocols: I2C, GPIO, MQTT

Deployment: 24/7 live production on AWS EC2

Tech: Python | Raspberry Pi | MQTT | I2C | GPIO | PWM
